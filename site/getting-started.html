Getting Started

Your first steps into AI Systems Engineering. No prior experience needed — just curiosity and a computer.

What You'll Need

1

A Computer

Any laptop or desktop with 8GB+ RAM works. For deep learning phases, we'll use free cloud GPUs (Google Colab) so you don't need expensive hardware.

2

Python Basics

If you know variables, loops, and functions, you're ready. If not, start with Phase 1 — we cover everything from scratch.

3

High School Math

Algebra and basic statistics are enough. We teach linear algebra and calculus concepts as we go, in context.

4

Patience

Building AI systems takes time. Expect to spend 2-4 hours per week per phase. The journey is the destination.

How This Works

This curriculum is built around the idea that learning by building beats passively consuming content. Each phase includes:

  • Concepts — Plain-english explanations of the theory
  • Code — Build algorithms from scratch before using frameworks
  • Business Context — How each concept applies to real business problems
  • Quizzes — Test your understanding with instant feedback
  • Playground — Experiment with code right in your browser

💡 Pro Tip

Don't rush through phases. Understanding > completion. If a concept doesn't click, try implementing it yourself before moving on.

Your First 5 Minutes

1

Try the Playground

Head to the Python Playground and run your first code snippet. No installation needed — it runs entirely in your browser using Pyodide.

2

Explore Phase 1

Start with Business Foundations. Click on a phase card to see its lessons, then click any lesson to dive in.

3

Track Your Progress

Click the '+' button on any lesson to mark it complete. Your progress is saved automatically in your browser.

4

Take a Quiz

Test what you've learned with our built-in quizzes. Instant feedback, no pressure.

Recommended Path

While you can jump around, we recommend this order:

  1. Phases 0-3 — Fundamentals (Python, stats, ML basics)
  2. Phases 4-6 — Core AI (Deep learning, NLP, transformers)
  3. Phases 7-9 — Systems (MLOps, cloud, agents)
  4. Phases 10-14 — Specialized (Vision, time series, recommenders)
  5. Phases 15-19 — Advanced (GenAI, ethics, capstone)
