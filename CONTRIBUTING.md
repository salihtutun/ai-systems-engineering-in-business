# Contributing to AI Systems Engineering in Business

Thank you for your interest in contributing! 🎉

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3. **Make your changes**
4. **Commit** (`git commit -m 'Add amazing feature'`)
5. **Push** (`git push origin feature/amazing-feature`)
6. **Open a Pull Request**

## Content Contributions

### Adding a Lesson

1. Create a new directory under the appropriate phase
2. Include:
   - `README.md` — lesson documentation
   - Code examples (`.py`, `.ipynb`)
   - `quiz.json` — optional quiz questions
   - `assets/` — images, diagrams

### Style Guide

- Use clear, business-contextualized explanations
- Include real-world examples and datasets
- Add interactive elements where possible
- Keep code runnable and well-commented
